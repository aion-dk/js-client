CastVoteRecord

CastVoteRecord: ContestMap<string>

A structure that contains the choice(s) of a voter.

Key is an electionId. Value is the chosen option represented by a handle-string

Example of a CastVoteRecord:

{
   '1': 'option1',
   '2': 'optionA'
}
