# Assembly Voting JS Client

Javascript API for building voter-facing election applications.

## Testing

To run tests:

```
npm run test
````

To run the tests in watch mode:

```
npm run tdd
```

To generate HTML documentation for external usage:

```
npm run html_docs
```
